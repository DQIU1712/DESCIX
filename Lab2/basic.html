¿Qué es Javascript y en qué se asemeja a Java?

JavaScript es un lenguaje de Scripting multiplataforma y orientado a objetos dentro de un ambiente host, que puede conectarse y proporcionar control programático sobre ellos. El javascript se asemeja a java en que utilizan llaves para delimitar bloques de códigos, ambos usan punto y comas como separador de instrucciones y tambien utilizan estructura de control de flujo como else/if,for, while,etc.
